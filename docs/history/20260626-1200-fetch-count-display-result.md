## 変更内容

### `views/main_page.py`

- `render()` 内、取得済みDFを取り出した直後（`df = st.session_state["loaded_df"]` の後）に
  取得件数サマリ表示ブロックを追加（取得後は常時表示）。
  - `n_rows = len(df)` / `n_stores = df["店舗名称"].nunique() if n_rows else 0`。
  - 0件: `st.warning("取得結果: 0件 — {企業} / 半径{半径}km に該当するデータが
    ありませんでした（取得処理は成功しています）")`。
  - 非0件: `st.success("取得結果: {n_rows:,}件（店舗数: {n_stores:,}）")`。
- 桁区切り（`{:,}`）で大規模データでも読みやすくし、0件＝失敗ではないことを文言で明示。
- 施設リスト側の0件 warning（店舗選択後・表示半径での絞込結果、「該当する推進園がありません」）は
  目的が異なるため残置。

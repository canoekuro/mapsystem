# 計画: 凡例の削除と距離表示の小数第2位化（issue 202606291519）

- 日時: 2026-06-29 15:19
- テーマ: issue-202606291519
- 対応する結果: [20260629-1519-issue-202606291519-result.md](20260629-1519-issue-202606291519-result.md)
- 起点 issue: `docs/issues/202606291519.md`

## 背景・目的

issue 202606291519 で次の2点が要望された。

1. 地図と一緒に表示される「保育園 / 幼稚園 / こども園」の凡例（色分けの説明）が、
   実データに推進園区分の実フラグが無く全マーカーがフォールバック色（灰）になるため
   意味をなしていない。→ 凡例を削除する。
2. 施設リスト下の距離が `約0.30202208520085133km` のようにフル精度で表示される。
   → `約0.30km` のように小数第2位まで丸める。

## 対象ファイルと変更内容

### A. 凡例の削除（画面の地図 + ダウンロードPNG）
- `lib/map_builder.py`: 定数 `_LEGEND_HTML` と、`build_map` 末尾での
  `m.get_root().html.add_child(...)` 挿入を削除。docstring を凡例なしに更新。
- `lib/static_map.py`: `render_static_map` 内の `_draw_legend(base)` 呼び出しと
  関数 `_draw_legend` を削除。モジュール／関数 docstring を凡例なしに更新。
- 色分け定数（`_FACILITY_COLORS` / `_FALLBACK_COLOR` / `_facility_color`）は残置。
  実データでは全件フォールバック灰になるため無害。バッジ／マーカーの単色化は行わない。

### B. 距離を小数第2位へ丸める（表示3箇所）
- `views/main_page.py`: 施設リスト `約{distance}km` → `約{distance:.2f}km`。
- `lib/map_builder.py`: マーカー tooltip `（{distance}km）` → `（{distance:.2f}km）`。
- `lib/png_builder.py`: PNG 施設リスト `約{row['距離km']}km` → `約{row['距離km']:.2f}km`。

### C. SPEC.md 整合
- §4.2: `推進園区分` を任意扱いにし「実データに区分フラグが無く色分け・凡例は出さない」旨、
  `距離km` を「表示時に小数第2位へ丸める」旨へ更新。
- §6.1.2: 色分けにフォールバック行を追記し凡例の項を削除。tooltip／施設リストの距離表記を
  `:.2f`（小数第2位）に更新。
- §8.3: PNG 施設カードの距離表記を `:.2f`（小数第2位）に更新。

### D. 記録
- 本 plan / result を `docs/history/` に保存し、`CHANGELOG.md` の `[Unreleased]` に追記。

## 検証方法
- 構文チェック（AST parse）と距離整形の確認（`約0.30km` になること）。
- 依存（folium/streamlit）が無い環境のため UI 実起動は未実施。

# 結果: 推進園区分の色分けを区別しやすい配色へ変更

- **日時:** 2026-07-07 09:36
- **対応 plan:** `docs/history/20260707-0936-color-contrast-improvement-plan.md`

## 1. 変更内容

`lib/colors.py`（唯一の色定義）と `SPEC.md` §6.1.2 の色表を更新。

| 推進園区分 | 変更前 | 変更後 |
|---|---|---|
| 認可保育所 | `#22C55E`（緑） | `#2A78D6`（青） |
| 認定こども園 | `#F59E0B`（黄） | `#EB6834`（橙） |
| 幼稚園 | `#EF4444`（赤） | `#4A3AA7`（紫） |
| 上記以外 | `#6B7280`（灰） | 変更なし |

色定義は一元化済みのため、地図（folium）・ダウンロードPNG・静的PNG・施設リスト・凡例の
すべてに `lib/colors.py` の変更が自動反映される（他ファイルの変更は不要）。

紛らわしかった認可保育所・認定こども園は、最も分離の大きい色対（青↔橙）へ割り当てた。

## 2. 検証結果

- `validate_palette.js "#2A78D6,#EB6834,#4A3AA7" --mode light`: Lightness band /
  Chroma floor / CVD separation / Contrast vs surface すべて **PASS**。
- 全ペアの CVD 分離（目標 ≥12）:
  - 青↔橙（認可保育所↔認定こども園）: ΔE 96.7 — 最大分離。
  - 橙↔紫: ΔE 96.7 超。
  - 青↔紫: ΔE 16.6（floor 12 以上、PASS）。通常色覚での ΔE は 32.6 で明確に区別可能。
- 白抜き番号のコントラスト（白文字 vs 背景色）: 青 4.42:1 / 橙 3.20:1 / 紫 8.56:1
  （太字数字に十分）。
- タイルモックで合成PNGを実描画し、3区分が判別しやすく凡例・マーカー・施設リストのバッジ色が
  連動することを目視確認。

## 3. 未対応事項

- 幼稚園の紫（`#4A3AA7`）はアプリのアクセント色（半径円・帯の `#7C3AED`）と同系。マーカーは
  白縁＋白抜き番号で判別可能だが、気になる場合はアクセント色側の調整を別途検討可能。
- Databricks Apps 実機での最終目視は環境上不可（ロジックはモックで実描画確認済み）。

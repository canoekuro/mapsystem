# UI微調整 計画（並び順・ボタン名・マップリセット）

- 日時: 2026-06-24 08:00
- テーマ: 企業一括DLの並び順/ボタン名変更、マップのリセットボタン追加
- ブランチ: `main`（`github.com/canoekuro/mapsystem`）

## 目的（ユーザー直接要望）

1. 企業一括ダウンロードの並びを「データダウンロード → 都道府県で絞り込み → 画像をダウンロード」に。
2. データボタン名を「企業一括データダウンロード」→「データダウンロード」に。
3. マップの誤操作対策として「マップをリセット」ボタンを追加し、押すと初期表示へ戻す。

## 対象ファイル
- `views/main_page.py`:
  - expander 内を `st.container()` 3つ（data/pref/image）で表示順を固定。先に都道府県を読み取り、
    データ/画像はその値を使う（都道府県絞込CSVの挙動は維持）。
  - データボタン名を「データダウンロード」に。
  - 地図カラムに「マップをリセット」ボタン。`st_folium(..., key=f"map_{store}_{radius}_{nonce}")` の
    nonce を押下で増やし再マウント（初期位置/ズームへ復帰）。
- `SPEC.md`（§6.1.2 リセット、§6.2 並び順・ボタン名）、`CHANGELOG.md`、`docs/history`。

## 役割分担
- メイン直接（小さなUI変更のため）: 上記すべて。各変更後に起動・スクショで確認。

## 検証
- `streamlit run app.py` 起動＋スクショで expander の並びとボタン名を確認。
- 店舗選択時に「マップをリセット」ボタンが出ること、押下で key 変化により地図が初期化されること。
- Databricks 再デプロイ。

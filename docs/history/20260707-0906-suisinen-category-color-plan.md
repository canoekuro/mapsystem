# 計画: 推進園区分による地図の色分け実装

- **日時:** 2026-07-07 09:06
- **対象 issue:** `docs/issues/推進園区分の色分け.md`
- **テーマ:** `推進園区分` 列の実値（`認可保育所` / `認定こども園` / `幼稚園`）に応じた色分け

## 1. 背景・目的

`推進園区分` 列は実データでは `認可保育所` / `認定こども園` / `幼稚園` の3値を取る。
一方、色分けテーブル（`_FACILITY_COLORS`）は旧区分名 `保育園` / `幼稚園` / `こども園`
をキーにしており、`認可保育所` と `認定こども園` がフォールバック色（灰）に落ちて
色分けが機能していなかった。実値に合わせて色分けを有効化する。

> 補足: 2026-06-29 の issue（202606291519）では「実データに区分フラグが無い」ため
> 凡例を削除し全マーカーをフォールバック色とした。今回の issue で実値が存在することが
> 判明したため、色分けを実データに整合させる。凡例は引き続き表示しない（issue の要求は
> 色分けのみ）。

## 2. 変更内容

### 色分けキーの実値化（同色を維持）

| 推進園区分 | 色 |
|---|---|
| 認可保育所 | `#22C55E`（緑、旧「保育園」） |
| 認定こども園 | `#F59E0B`（黄、旧「こども園」） |
| 幼稚園 | `#EF4444`（赤、変更なし） |
| 上記以外 | `#6B7280`（灰、フォールバック） |

### 対象ファイル

1. `lib/map_builder.py` — インタラクティブ地図（folium）の `_FACILITY_COLORS`
2. `lib/png_builder.py` — ダウンロードPNGの `_FACILITY_COLORS`
3. `lib/static_map.py` — 静的PNG（RGBタプル）の `_FACILITY_COLORS`
4. `views/main_page.py` — 施設リストのバッジ色 `_FACILITY_COLORS`
5. `scripts/gen_sample.py` — サンプル生成。表示名の末尾語（保育園/幼稚園/こども園）は
   自然な名称として残しつつ、`推進園区分` は実値へマッピング
6. `data/master.csv` — 上記スクリプトで再生成
7. `SPEC.md` — §4.2（推進園区分の必須区分・説明）と §6.1.2（色分け表）を実態へ整合

## 3. 検証方法

- `python3 scripts/gen_sample.py` で `master.csv` を再生成し、`推進園区分` が
  `認可保育所` / `認定こども園` / `幼稚園` の3値のみになることを確認。
- AST で4ファイル + 注釈付き代入の `_FACILITY_COLORS` が3実値をキーに持つことを確認。
- 施設名（推進園名称）は `〜保育園` 等の自然名を維持していることを確認。

# 計画: 推進園区分の色分けを区別しやすい配色へ変更

- **日時:** 2026-07-07 09:36
- **対象:** Databricks 実機確認でのフィードバック
- **テーマ:** 認可保育所（緑）と認定こども園（黄）が紛らわしいため、区別しやすい配色へ

## 1. 背景・目的

先行対応（PR #10）で色分けと凡例を実装したが、実機で
**認可保育所（`#22C55E` 緑）と認定こども園（`#F59E0B` 黄）の違いが分かりにくい**との指摘。
色相が近く彩度も高いため隣接時に判別しづらい。区別しやすく、かつ色覚多様性にも配慮した
高コントラストな配色へ変更する。

## 2. 変更内容

`lib/colors.py` の `FACILITY_COLORS` を、青/橙/紫の3色へ変更する（白抜き番号が読めるよう
各色は十分に濃い）。紛らわしかった2区分は最も離れた色対（青↔橙）へ割り当てる。

| 推進園区分 | 変更前 | 変更後 |
|---|---|---|
| 認可保育所 | `#22C55E`（緑） | `#2A78D6`（青） |
| 認定こども園 | `#F59E0B`（黄） | `#EB6834`（橙） |
| 幼稚園 | `#EF4444`（赤） | `#4A3AA7`（紫） |
| 上記以外 | `#6B7280`（灰） | 変更なし |

色は dataviz スキルの `validate_palette.js` で検証したうえで選定する
（隣接CVD ΔE ≥ 12、対比 ≥ 3:1 を満たすこと）。

対象ファイル: `lib/colors.py`（一元定義のためここのみ）、`SPEC.md` §6.1.2 の色表。

## 3. 検証方法

- `validate_palette.js "<3色>" --mode light` で Lightness band / Chroma floor /
  CVD separation / Contrast vs surface がすべて PASS することを確認。
- 全ペア（青↔橙 / 青↔紫 / 橙↔紫）の CVD 分離が目標（12以上）を満たすことを確認。
- タイルモックで地図PNG・合成PNGを実描画し、3区分が判別しやすく白抜き番号が読めることを目視確認。

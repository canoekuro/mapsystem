## 変更内容

- `views/main_page.py`:
  - 企業一括ダウンロード expander の表示順を `st.container()` 3つで固定し、
    **データダウンロード → 都道府県で絞り込み → 画像をダウンロード** に変更。
    都道府県の選択値を先に読み取り、データ/画像の両方に反映（都道府県絞込CSVの挙動は維持）。
  - データボタン名を「企業一括データダウンロード」→「**データダウンロード**」に変更。
  - 地図カラムに「**マップをリセット**」ボタンを追加。`st_folium` の `key` に
    `map_{store}_{radius}_{nonce}` を用い、押下で `st.session_state["map_nonce"]` を増やして
    再マウントし、`build_map` の初期位置/ズームへ復帰させる。
- `SPEC.md`（§6.1.2 リセットボタン、§6.2 並び順・ボタン名）、`CHANGELOG.md` を更新。

- マップのデータ出典表示の下に、以下のSQLの実行結果の表を追加したい。また、この表もダウンロードできるようにしたい。
```
%sql
SELECT
  `店舗名称`, 
  `店舗コード`, 
  `店舗_都道府県`, 
  COUNT(DISTINCT `推進園名称`) AS `推進園数`
FROM
  kh_aigrd_databricks_prd_01.default.immune_promo_nursery
WHERE
 `企業名称` <= UI上で選択されている企業名称
 `距離km` <= UI上で選択している取得半径
GROUP BY
  `企業名称`, 
  `店舗名称`, 
  `店舗コード`, 
  `店舗_都道府県`
```

- 推進園の区分をなくし、全て同一の色（"#7030A0"）で表示する。併せて、凡例表記を消す。
- 「小売店名 周辺マップ概要 ｜ 半径＊km圏内施設リスト」の帯と、「施設リスト」の間が空いている。これを詰めたい。
- マップと「施設リスト」の間が空いている。今のマップサイズを前提として、もう少し隙間を詰めたい。（左詰めにする）
- 「施設リスト」は1列10件として、それ以上の場合は列を増やしていく形にする。列が3列以上にわたる場合には横スクロールバーを出す。
- 施設リストの下に、当年実績ケース数と前年比を追加する。※変更については、docs/issues/image1.pngを参照。

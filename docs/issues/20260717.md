# 周辺マップ概要 レイアウト微調整（image2.png）

- 施設リストの枠線（罫線）はいらない。
- 「施設リスト」帯の上に、もうちょっと隙間を開けたい。
- マップと「施設リスト」の間の隙間を、もっと狭めたい。
- 施設リストの3列目が見切れている。初期状態で見切れが発生しないよう、2列で「施設リスト」の帯の幅に
  なるように調整する（3列目以降はスクロールすれば見られる、でOK）。
